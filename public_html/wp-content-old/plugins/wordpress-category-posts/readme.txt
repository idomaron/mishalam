== Installation ==
1. Copy the wp-category-posts file to /wp-content/plugins/
1. Activate the plugin at your blog's Admin -> Plugins screen


== USAGE ==

<?php wp_cat_posts(3); ?>

Insert this line in where you want to use it, substituting the number for the ID of category you want to display.

If you use a plugin like PHP Exec you can easily use the WP Category Posts plugin inside of a WordPress "page" to create a specific archive page.
